# CryptoAtlas

Cryptocurrency tracker and news app.

## Demo

https://cryptoatlas.angeltasevski.com/

## Run Locally

Clone the project

```bash
git clone https://github.com/tupacan/crypto-atlas
```

Go to the project directory

```bash
cd crypto-atlas
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm start
```
